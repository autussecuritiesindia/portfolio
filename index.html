🐙 GitHub Pages Setup — Free Forever (5 min)
1
Create a free GitHub account
Go to github.com → Sign up (free) → Verify email.
2
Create a new repository
Click + → New repository, name it portfolio, set to Public, tick ✅ Add a README file, then Create repository.
3
Upload the dashboard file
In repo → Add file → Upload files → drag index.html → Commit changes.
4
Enable GitHub Pages
Settings → Pages → Source: Deploy from branch → Branch: main / (root) → Save.
5
Get your permanent URL
Wait ~60s → your URL: https://YourUsername.github.io/portfolio — bookmark it!
6
Update Google Sheet data
Just update the sheet → tap 🔄 Sync Sheet → dashboard auto-refreshes. No file uploads needed!
